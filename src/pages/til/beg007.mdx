If you write `/*` at the beginning and `*/` at the end of a block of text, the text becomes a comment.

Use the **copy** button in the code box below and paste it to the console.

```javascript
/*
      _____                    _____                    _____  
     /\    \                  /\    \                  /\    \ 
    /::\    \                /::\    \                /::\____\
    \:::\    \               \:::\    \              /:::/    /
     \:::\    \               \:::\    \            /:::/    / 
      \:::\    \               \:::\    \          /:::/    /  
       \:::\    \               \:::\    \        /:::/    /   
       /::::\    \              /::::\    \      /:::/    /    
      /::::::\    \    ____    /::::::\    \    /:::/    /     
     /:::/\:::\    \  /\   \  /:::/\:::\    \  /:::/    /      
    /:::/  \:::\____\/::\   \/:::/  \:::\____\/:::/____/       
   /:::/    \::/    /\:::\  /:::/    \::/    /\:::\    \       
  /:::/    / \/____/  \:::\/:::/    / \/____/  \:::\    \      
 /:::/    /            \::::::/    /            \:::\    \     
/:::/    /              \::::/____/              \:::\    \    
\::/    /                \:::\    \               \:::\    \   
 \/____/                  \:::\    \               \:::\    \  
                           \:::\    \               \:::\    \ 
                            \:::\____\               \:::\____\
                             \::/    /                \::/    /
                              \/____/                  \/____/ 
*/
console.log("Comments can be creative!");
```
