Instead of writing the same text many times, we store it in a variable and then  we use the variable instead of the text.

For that, we:

1. Declare a variable<br/>
2. Store the text on the variable (see next TIL)

To declare a variable, you write `var` followed by a name.

```javascript
// The variable someText is being declared on the line below.
var someText;
```
