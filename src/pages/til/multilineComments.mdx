If you write `/*` at the beginning and `*/` at the end of a block of text, the text becomes a comment.

```javascript
/*
 As per
⊂_ヽ
　 ＼＼   my
　　 ＼( ͡° ͜ʖ ͡°)
　　　 >　⌒ヽ
　　　/ 　 へ＼
　　 /　　/　＼＼last
　　 ﾚ　ノ　　 ヽ_つ
　　/　/
　 /　/|
　(　(ヽ
　|　|、＼email
　| 丿 ＼ ⌒)
　| |　　) /
ノ )　　Lﾉ
(_／ 
*/
console.log("Comments can be creative!");
```
